PERSONAL STATEMENT

개발을 즐기고 특히 웹을 좋아합니다. 웹 개발을 처음하기 시작한 것은 2011년 입니다. 학교 전공 수업인 웹프로그래밍을 수강하면서 시작하게 되었고, 수강 이후에 개인적으로 웹 개발이 재미가 있어서 지금까지 지속적으로 공부해오고 있습니다. Play! Framework, Ruby on Rails, jQuery, Bootstrap 등의 웹 기술에 관심을 두고 공부해왔고, 개인 프로젝트와 팀 프로젝트에 그 내용을 적용해보기도 했습니다.

지금은 언니의파우치 개발팀에서 웹 개발자로 근무하고 있습니다. 처음에 들어왔을때는 7명의 작은팀 이었는데 웹 개발자가 없었습니다. 현재 서비스 되고 있는 언니의파우치 홈페이지를 git init 부터 시작하여 지금까지 전담하여 주도적으로 개발하였습니다.

또 다른 경력으로는 미국 뉴저지에 있는 United Merchant Services 라는 회사에서 인턴으로 근무하였습니다. 이 기간동안 .NET MVC3 기반의 사내 인트라넷 서비스 개발과 Ruby on Rails 기반의 회사 홈페이지를 제작하는 프로젝트를 한 경험이 있습니다. 이 때 제작한 홈페이지가 바로 UPSolution 입니다. 당시 CTO 분과 함께 2인이 개발한 웹사이트 입니다. 상무님은 프로젝트 매니저 역할을 하였기 때문에 대부분의 코드를 직접 작성하였습니다.

새로운 기술을 배우는 것을 좋아하고 좋아하는 만큼 배우는 속도도 빠릅니다. UPSolution 홈페이지를 개발할 당시 Ruby On Rails 의 기본 튜토리얼만 아는 상태에서 시작했는데 개발 시작 3개월만에 서비스를 런칭할 수 있었습니다. 또한 데이터베이스 설계 부터 UI 구현까지. 백엔드, 프론트엔드 가리지 않고 풀스택으로 개발할 수 있습니다. 해외 인턴십 당시 테이블 100개 이상의 복잡한 규모의 데이터베이스를 다룬 경험이 있습니다. MSSQL 과 .NET MVC3 로 만들어진 인트라넷 서비스로서 하루 수백만건의 카드 결제 데이터가 데이터베이스에 쌓이는데 이를 가지고 stored procedure, functions 과 같은 기술을 활용하여 복잡한 연산을 처리한 경험이 있습니다.
